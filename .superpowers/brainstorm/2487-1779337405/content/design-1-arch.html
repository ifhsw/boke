设计第1部分：整体架构变更

从 Markdown textarea → TipTap 富文本编辑器

内容存储变化

层面 改前 改后
编辑器 <textarea> + marked 预览 TipTap React 组件 (WYSIWYG)
存储格式 Markdown 字符串 TipTap JSON (Prisma Json 类型)
服务端渲染 compileMdx() → MDX组件 generateHTML() → sanitized HTML
MDX 组件 Callout, CodeBlock, ImageCaption 重写为 TipTap Node Extensions

新增/变更的依赖

包 用途
@tiptap/reactReact 编辑器组件
@tiptap/starter-kit基础扩展包 (bold, italic, heading, code, etc.)
@tiptap/extension-image图片拖拽/粘贴/上传
@tiptap/extension-code-block-lowlight代码块 + Shiki 高亮
@tiptap/extension-table表格支持
@tiptap/extension-link链接
@tiptap/extension-placeholder占位文字
@tiptap/extension-character-count字数统计
@tiptap/extension-task-list + task-item任务列表
lowlightShiki → lowlight 适配 TipTap

Prisma Schema 变更
